# Система управления банковскими картами

Приложение автоматизирует операции связанные с банковскими картами для пользователей и администраторов.

## Функционал для пользователей: ##
- Просмотр своих карт
- Просмотр баланса карты
- Перевод между счетами

## Функционал для администратора: ##
- Управление пользователями (создание, удаление и т.д.)
- Управление картами (создание, удаление, активация, блокировка и т.д.)
- Просмотр всех карт и пользователей

## Используемые технологии: ##
- Java 17
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- PostgreSQL
- Docker
- Liquibase
- OpenAPI


## Инструкция запуска ##

### 1. Клонирование репозитория: ###
```bash
git clone https://github.com/Zaharrkaa/card_manager_rest.git
```
